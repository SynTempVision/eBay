### Purpose: 
- To define the standardized process for creating accurate, complete, and professional eBay listings.
- To ensure listings are consistent, traceable, and aligned with SynTemp Vision inventory control procedures.
- 
This SOP applies to:
```
- Identified items to be listed on eBay
```
---
### Definitions
##### Draft Listing: 
- A completed but unpublished eBay listing pending internal approval.
##### Listing Approval:
- Internal confirmation required before publishing a listing live on eBay.

üîπ Step 1: Photography Requirements
- First Photo (Primary Image):
    - Entire item shown
    - Powered ON (if applicable)
    - Video If needed
    - All included parts/components visible
- Required Angles:
    - Front
    - Back
    - Sides
    - Top
- Close-ups of:
    - Damage
    - Missing components
    - Wear and tear
    - Dents or blemishes
- Identification Photos
    - Model number
    - Serial number (if allowed)
    - Manufacturer label
- Remove or obscure:
    - Old auction stickers
    - University labels
    - Lab property tags
    - Any source-identifying labels
- Image Quality
    - Proper lighting
    - No heavy shadows or dead space
    - Background clean and consistent
    - Crop and straighten images as needed
- Special Category: Burnishing Tools & Boring Bars
    - Must be photographed on green cutting mat
    - Include:
        - Overall length
        - Diameter
        - NPT size
        - Shaft size
        - Attachment model numbers

üîπ Step 2: Title Creation
- Title must include (when applicable):
    - Brand
    - Part name
    - Model number(s)
    - Part number
    - Manufacturer date or expiration date (if relevant)
    - Key specs:
        - Length
        - Diameter
        - NPT size
        - Shaft size
        - Attachments or accessory names
üîπ Step 3: Custom SKU
- Enter SKU in eBay "Custom Label" field.
- SKU must appear in at least one listing photo.
- SKU must match InvenTree record.

üîπ Step 4: Item Category
- If using ‚ÄúHave one to sell? Sell now‚Äù:
- Verify auto-selected category is correct.
- Otherwise:
    - Manually select most accurate category.
    - Category must reflect primary function of item.

üîπ **Step 5: Item Condition**

- **Used**
    - Use standard verbiage:
      > Pulled from a working environment. Item has some usual wear-and-tear scuffs and blemishes but is overall in good condition. See pictures for more detail.

- **New**
    - Include ‚ÄúNEW‚Äù in title if space allows.
    - Specify unopened or sealed status in description if applicable.

- Condition must match photo evidence.

##### Rules & Constraints
- No unnecessary filler words
- No ALL CAPS except NEW when applicable
- Maximize keyword relevance
